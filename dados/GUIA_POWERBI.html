Medidas DAX

Modelagem → Nova Tabela → digitar Medidas = {0} → adicionar cada medida abaixo

Medidas Básicas
Receita Total = SUM(fato_vendas[valor_pedido])

Qtd Pedidos = COUNTROWS(fato_vendas)

Ticket Medio = DIVIDE([Receita Total], [Qtd Pedidos], 0)

Clientes Ativos = DISTINCTCOUNT(fato_vendas[documento])
Medidas Temporais
Receita Mes Anterior =
CALCULATE(
    [Receita Total],
    DATEADD(serie_temporal[ano_mes_dt], -1, MONTH)
)

Variacao MoM =
DIVIDE(
    [Receita Total] - [Receita Mes Anterior],
    [Receita Mes Anterior], 0
)

Receita Acumulada Ano =
CALCULATE(
    [Receita Total],
    DATESYTD(serie_temporal[ano_mes_dt])
)
Segmentação e Clientes
Clientes VIP =
CALCULATE(
    DISTINCTCOUNT(dim_clientes[documento]),
    dim_clientes[segmento] = "VIP"
)

Clientes Em Risco =
CALCULATE(
    DISTINCTCOUNT(dim_clientes[documento]),
    dim_clientes[segmento] IN {"Em Risco", "Perdendo VIP"}
)

Pct Omnichannel =
DIVIDE(
    CALCULATE(DISTINCTCOUNT(dim_clientes[documento]), dim_clientes[omnichannel] = TRUE()),
    DISTINCTCOUNT(dim_clientes[documento]), 0
)
Ranking e Comparação (Página Vendedor)
Meu Ranking =
RANKX(
    ALL(dim_vendedores[vendedor]),
    CALCULATE([Receita Total]),
    , DESC, Dense
)

Media Equipe =
DIVIDE(
    CALCULATE([Receita Total], ALL(fato_vendas[vendedor])),
    CALCULATE(DISTINCTCOUNT(fato_vendas[vendedor]), ALL(fato_vendas[vendedor])),
    0
)

Montar as Páginas

Criar 4 páginas no Power BI e adicionar os visuais conforme abaixo

Página 1

CEO — Visão Executiva

  • 5 Cartões: Receita Total, Qtd Pedidos, Ticket Médio, Clientes Ativos, Variação MoM
  • Gráfico de Linhas: serie_historico_previsao → eixo: ano_mes_dt, valor: receita_total, legenda: tipo
  • Gráfico de Barras: fato_vendas → eixo: trimestre, valor: [Receita Total]
  • 3 Cartões Previsão: previsao_proximos_meses → mes + receita_prevista
  • Segmentação: fato_vendas[ano]
Página 2

Marketing — Clientes e Canais

  • 4 Cartões: Clientes Ativos, Clientes VIP, Clientes Em Risco, % Omnichannel
  • Rosca: dim_clientes → legenda: segmento, valor: COUNT(documento)
  • Barras Horizontais: fato_vendas → eixo: canal, valor: [Receita Total]
  • Treemap: fato_vendas → categoria: faixa_valor, tamanho: [Qtd Pedidos]
  • Tabela: dim_clientes → nome, segmento, frequencia, monetario, ticket_medio
Página 3

Performance — Gerente de Loja

Dicas Importantes

Configurações que fazem diferença no resultado final

A

Gráfico de Linhas — desativar hierarquia de datas

No campo ano_mes_dt, clicar na seta ao lado do campo no eixo X e selecionar "ano_mes_dt" diretamente (não "Hierarquia de datas"). Isso garante que cada mês apareça individualmente.

B

Formatação Condicional — Variação MoM

No cartão de Variação MoM: Formatar → Cor do valor → Por regras: se valor ≥ 0 → cor #00C9A7 (teal), se valor < 0 → cor #E53E3E (vermelho).

C

Sync Slicers — filtros sincronizados

Para que o filtro de Ano funcione em todas as páginas: Exibir → Sincronizar segmentações de dados → marcar todas as páginas para o segmentador de Ano.

D

Cores do tema

Exibir → Temas → Personalizar tema: Principal #4A1E6B · Destaque #00C9A7 · Positivo #00C9A7 · Negativo #E53E3E · Previsão #FF8F00

E

Decimal com vírgula (formato BR)

Os arquivos CSV usam vírgula como decimal e ponto e vírgula como separador. No Power Query, confirmar que os números foram reconhecidos corretamente. Se aparecer como texto, trocar o tipo para Número Decimal Fixo com localidade Português (Brasil).
