O que foi feito

Do dado bruto ao dashboard interativo, passando por tratamento, modelagem e automacao.

🧹 Tratamento de Dados

Limpeza de dados de teste, deduplicacao de clientes, padronizacao de nomes e valores, unificacao de 3 fontes com 99,98% de match entre pedidos e clientes.

📐 Modelagem Star Schema

Tabela fato de vendas com dimensoes de clientes, vendedores e tempo. Extensivel para novas dimensoes sem alterar a estrutura existente.

🧠 Segmentacao RFM

Classificacao de clientes em 7 segmentos (VIP, Leal, Regular, Novo Promissor, Em Risco, Perdendo VIP, Inativo) usando rank percentual.

📈 Modelo Preditivo

SARIMA(1,1,1)(1,1,1,12) selecionado entre 3 modelos, com MAPE de 10,5%. Captura sazonalidade anual e gera previsao para 3 meses.

🤖 Automacao Diaria

GitHub Actions roda o pipeline Python todo dia as 08h (BRT), retreina o modelo e atualiza o dashboard automaticamente.

🌐 Dashboard Web

HTML + Chart.js hospedado no GitHub Pages. Acessivel por qualquer navegador, sem instalacao, responsivo e com tema escuro.
